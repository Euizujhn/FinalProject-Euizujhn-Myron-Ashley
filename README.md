be sure to save it to your favorites to find it easier._

# <ins> Build and Run Instructions: </ins>
-Press the Green Code Button
-Download ZIP
-Find in folder and Unzip
-Open Intellij and Open the project
-IF Error (Error: JavaFX runtime components are missing, and are required to run this application) appears
Go to the 4 bars, hover over run, edit configurations, modify options, add VM, 
enter --module-path "path/to/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml,javafx.graphics,javafx.base
